# Insights（可选）

本目录用于存放**可再生成**的分析产物（例如对上游 SeaTunnel commits 的统计洞察）。

## 如何使用洞察

建议把洞察结论反哺到：

- `skills/seatunnel-contributor-playbook/references/CHANGE_RECIPES.md`
- 或新增更细粒度的 skills（如按 connector 类型拆分）
